# Recording UI Progress — 2026-03-24

## Session Summary

Built mic/speaker audio level monitoring and webcam preview for the recording panel overlay.

## What Was Done

### 1. Mic Level Monitoring (PipeWire)
- **Problem**: GStreamer `pulsesrc`/`autoaudiosrc` couldn't capture mic audio on GNOME Wayland/PipeWire system
- **Fix**: Replaced GStreamer with native PipeWire `pw_stream` for audio capture
- **Approach**: Generic `start_audio_level_stream()` function that creates a PipeWire stream, computes peak level from audio samples, stores in atomic
- **UI**: Orange 5-bar VU meter on Mic tile, responds to voice input

### 2. System Audio Monitoring (Speaker)
- **Approach**: Second PipeWire stream with `stream.capture.sink=true` to tap system audio output digitally
- **Differentiation**: Mic = warm orange bars, Speaker = cool blue/teal bars
- **Audio processing**: Mic uses peak detection, Speaker uses RMS averaging for natural variation
- **Noise gate**: Mic stream ignores levels below 0.15 to avoid picking up speaker bleed (acoustic feedback on laptop)

### 3. D-Bus Communication
- **Problem**: zbus→Qt5 D-Bus signal delivery was broken (signals sent but never received by overlay)
- **Fix**: Switched to polling approach — overlay timer (33ms) calls `GetMicLevel` and `GetSpeakerLevel` D-Bus methods
- **Daemon methods**: `get_mic_level()` and `get_speaker_level()` read from static atomics

### 4. Webcam Preview Tile
- Renamed existing `Record` tile to `Webcam` in Row 2
- Click toggles webcam on/off (shows tick indicator)
- Right-click opens context menu with:
  - **Camera**: None, detected v4l2 devices (filtered, deduplicated)
  - **Size**: Small (120×160), Medium (200×260), Large (280×370), Huge (360×480), Fullscreen
  - **Shape**: Circle, Square, Rectangle, Vertical
  - **Options**: Flip Camera
- Auto-refreshes device list on each menu open (picks up DroidCam/OBS Virtual Camera)
- Live camera feed displayed via GStreamer pipeline (`v4l2src ! videoconvert ! appsink`)
- Frame drawn with shape clipping (circle, rounded rect, etc.)
