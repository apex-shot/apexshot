# Chrome Web Store Listing

## Store Listing Information

### Short Description (132 characters max)
Capture entire webpages with one click. Scroll, screenshot, and edit in ApexShot.

### Detailed Description

Capture entire webpages with one click - no more taking multiple screenshots and stitching them together manually. This extension automatically scrolls through the page, captures each section, and stitches them into a single high-quality screenshot that's sent directly to ApexShot for editing and sharing.

**Perfect for:**
- Long articles and blog posts that require scrolling
- Documentation pages and tutorials
- Social media feeds and threads
- Product pages with detailed information
- Any webpage content that doesn't fit on a single screen

**Key Features:**
- **One-click capture**: Just click the extension button and it does the rest
- **Automatic scrolling**: The extension handles scrolling through the entire page
- **Smart stitching**: Seamlessly combines multiple screenshots into one image
- **Direct integration**: Screenshots automatically open in ApexShot for editing
- **No manual setup**: Extension configures itself automatically

**How it works:**
1. Install ApexShot desktop app from your package manager
2. Install this Chrome extension
3. Navigate to any webpage
4. Click the extension button
5. The extension captures the entire page and sends it to ApexShot

**Why install this extension?**
- Save time: No more manual screenshot stitching
- Better quality: Automatic scrolling ensures consistent capture
- Seamless workflow: Direct integration with ApexShot editor
- Free and open source: No cost, no tracking, no data collection
- Privacy-focused: Screenshots are processed locally on your machine

**Requirements:**
- ApexShot desktop app (free, open source)
- The extension automatically configures itself on first use

### Additional Information

**Permissions Explained:**
- `activeTab`: Access to the current tab to capture screenshots
- `scripting`: Inject scripts to handle page scrolling and capture
- `nativeMessaging`: Communicate with ApexShot desktop app
- `notifications`: Show capture status and error messages
- `<all_urls>`: Allow capture on any webpage

**Privacy:**
- This extension does not collect or transmit any personal data
- Screenshots are sent locally to the ApexShot desktop app
- No data is sent to external servers
- No tracking or analytics
