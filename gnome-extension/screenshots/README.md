# Screenshots for extensions.gnome.org Submission

## Required Screenshots (3-5 images)

For the GNOME Extension submission, you need 3-5 screenshots showing the extension in action. Here are the recommended screenshots:

### 1. Screenshot Preview Stacking (REQUIRED)
- Show ApexShot with a screenshot preview window
- Drag the preview to another application (e.g., Discord, Slack, browser)
- Capture the moment when the preview stays on top while dragging
- This demonstrates the window stacking feature

### 2. Recording Mask (REQUIRED)
- Start a screen recording in ApexShot
- Show the dimmed recording mask around the selected area
- Capture the recording controls overlay
- This demonstrates the recording mask feature

### 3. Runtime Overlays (OPTIONAL)
- During recording, enable audio indicators
- Capture the mic/speaker status overlays
- This demonstrates the remaining runtime overlay features

### 4. Full Recording Session (OPTIONAL)
- Capture a full recording session with:
  - Recording mask
  - Timer display
  - Control buttons (pause, stop, mic)
  - Any enabled mic/speaker overlays

### 5. Extension Enabled (OPTIONAL)
- Show GNOME Extensions app with the extension enabled
- This proves the extension is installed and working

## How to Take Screenshots

1. Use ApexShot itself to capture the screenshots
2. Ensure high resolution (1920x1080 or higher)
3. Use PNG format for best quality
4. Name them descriptively: `01-preview-stacking.png`, `02-recording-mask.png`, etc.

## Upload Instructions

Once you have the screenshots:
1. Upload them to the screenshots directory
2. They will be included in the extension zip package
3. During submission to extensions.gnome.org, you'll upload them separately
